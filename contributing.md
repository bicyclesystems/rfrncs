# contributing to rfrncs

thank you for considering contributing to rfrncs!

## get started

1. fork the repository to your own github account
2. clone the repository:
   ```
   git clone https://github.com/bicyclesystems/rfrncs.git
   ```
3. create a branch locally with a descriptive name
4. make your changes to the csv files
5. commit and push changes to your fork
6. open a pull request

## csv file naming

all csv files should be lowercase with hyphens instead of spaces:
- `css-pseudo-classes.csv`
- `css-selectors.csv`
- `html-elements.csv`

## getting help

contributions are always welcome. PRs should:
- address a single concern
- include clear commit messages
- follow the existing csv structure
